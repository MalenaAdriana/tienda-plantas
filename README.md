Tienda de Plantas

Aplicación web full stack con carrito de compras y gestión de pedidos.

Tecnologías
- HTML
- CSS
- JavaScript
- Node.js
- Express

Funcionalidades
- Carrito dinámico
- Gestión de pedidos
- API REST
- Persistencia de datos

Cómo ejecutar
1. Instalar dependencias:
npm install

2. Ejecutar servidor:
node server.js

3. Abrir en navegador:
http://localhost:3000
